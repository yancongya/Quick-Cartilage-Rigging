# 快速入门教程

本教程将通过一个完整的实例，指导您从零开始，使用"快速软骨绑定"插件制作一个简单的尾巴或触手绑定。

**目标**: 将一根单独的骨骼，变成一个拥有FK控制器和弹性跟随效果的骨骼链。

---

### 第1步：创建基础骨架

1.  打开Blender，删除默认的立方体。
2.  按下 `Shift + A`，在菜单中选择 `骨架 (Armature)` > `单根骨骼 (Single Bone)`，创建一个新的骨架。
3.  选中该骨架，按下 `Tab` 键进入 **编辑模式 (Edit Mode)**。

![进入编辑模式](https://docs.blender.org/manual/en/latest/_images/sculpt-mode_sculpt-mode-toolbar.png)
*<p align="center">（示意图：确保您处于骨架的编辑模式）</p>*

### 第2步：细分骨骼链

现在，我们将使用插件把这根骨骼变成一串骨链。

1.  确保刚才创建的骨骼处于 **选中状态**。
2.  在右侧的 `Damped Track` 插件面板中，找到 **"1. 骨骼分割工具"** 区域。
3.  设置 **"段数"** 为 `8`（您可以根据需要调整）。
4.  点击 **`平均细分`** 按钮。

您会看到，原来的单根骨骼现在变成了一条由8段小骨骼组成的链条。

### 第3步：生成FK控制器

接下来，我们为这条骨链创建一套用于控制动画的FK控制器。

1.  执行完上一步后，插件会自动选中新骨链的第一节骨骼。
2.  在插件面板的 **"绑定设置"** 区域，点击 **`2. 生成FK绑定`** 按钮。

操作完成后，您会看到场景中出现了一串带有圆形自定义图形的控制器骨骼（名为 `ctr_...`）。这些就是我们的FK控制器。

### 第4步：应用软骨绑定

最后一步，让骨骼链"软"起来。

1.  按下 `Tab` 键，切换到 **姿态模式 (Pose Mode)**。
2.  此时，第一个控制器骨骼（`ctr_...001`）应该处于激活状态。
3.  在插件面板的 **"绑定设置"** 区域，点击 **`3. 生成软骨绑定`** 按钮。

插件会自动为骨骼链添加"阻尼追踪"约束，并设置好驱动器。

### 第5步：测试与调整

恭喜，绑定完成！现在来测试一下效果。

1.  在姿态模式下，尝试旋转任意一个圆形的FK控制器。
2.  您会发现，它后面的骨骼链会像尾巴一样产生柔软的跟随效果。
3.  选中第一个控制器（`ctr_...001`），在插件面板的 **"控制器属性"** 区域，您可以看到 **"难崩系数"** 的滑块。
4.  尝试拖动这个滑块，观察骨骼链的跟随效果变化。系数越高，尾巴越"硬"；系数越低，则越"软"。

---

至此，您已经掌握了插件最核心的工作流程。快去尝试用"斐波那契细分"和其他参数来创造更多有趣的效果吧！
