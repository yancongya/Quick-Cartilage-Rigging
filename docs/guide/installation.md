# 安装与激活

本章节将指导您如何在Blender中成功安装并激活"快速软骨绑定"插件。

## 下载插件

首先，确保您已经拥有插件的 `.py` 文件 (`Quick Cartilage Rigging.py`)。

## 安装步骤

Blender 插件的安装过程非常简单：

1.  打开 Blender。
2.  在顶部菜单栏中，选择 `编辑 (Edit)` > `偏好设置 (Preferences...)`。
3.  在弹出的窗口中，切换到 `插件 (Add-ons)` 选项卡。
4.  点击右上角的 `安装... (Install...)` 按钮。

    ![Install Button](https://docs.blender.org/manual/en/latest/_images/preferences_addons_install_button.png)

5.  在文件浏览器中，找到并选中您下载的 `Quick Cartilage Rigging.py` 文件，然后点击 `从文件安装插件 (Install Add-on from File)`。
6.  返回到插件列表后，它会自动筛选出刚刚安装的插件。找到名为 **"Rigging: 🦴快速软骨绑定"** 的条目。
7.  勾选插件名称左侧的复选框以启用它。

    ![Enable Addon](https://docs.blender.org/manual/en/latest/_images/preferences_addons_enable.png)

## 激活成功

激活后，您就可以在Blender的界面中找到插件的面板了。

*   **位置**: `3D 视图` > `侧边栏 (按 N 键调出)` > `Damped Track` 选项卡

现在，您可以开始使用插件了！
