# 界面概览

"快速软骨绑定"插件提供了两种主要的操作方式：一个位于3D视图侧边栏（N面板）的常驻面板，以及为了提升效率而设计的右键快捷菜单。

## 1. 主面板（N面板）

这是插件最主要的交互界面。

*   **如何访问**: 在3D视图中，按下 `N` 键打开侧边栏，然后找到名为 `Damped Track` 的选项卡。

面板根据您当前所处的模式（编辑模式或姿态模式）动态显示可用选项。

### 面板分区说明

1.  **模式切换与更新**:
    *   **对象/编辑/姿态**: 快速在三种常用模式间切换。
    *   **刷新版本**: 点击可检查插件是否有新版本。

2.  **1. 骨骼分割工具** (`仅在编辑模式下可用`):
    *   **段数/系数**: 设置细分骨骼的数量和（斐波那契细分）的分布系数。
    *   **斐波那契细分**: 按斐波那契数列比例细分骨骼，产生由密到疏的效果。
    *   **平均细分**: 将骨骼均匀地分割成指定段数。

3.  **绑定设置**:
    *   **2. 生成FK绑定** (`仅在编辑模式下可用`): 为选中的骨骼链创建一套FK控制器。
    *   **3. 生成软骨绑定** (`仅在姿态模式下可用`): 为骨骼链应用阻尼追踪约束，实现软骨效果。

4.  **控制器属性** (`仅在姿态模式下，并选中控制器骨骼时可用`):
    *   **难崩系数**: 控制阻尼追踪的强度，数值越高，骨骼链的跟随性越强，表现得越"Q弹"。
    *   **圆环缩放**: 调整FK控制器（圆环）的显示大小。
    *   **控制骨骼可见性**: 提供按钮快速显示/隐藏全部或仅显示根部控制器，方便动画调节。

## 2. 右键快捷菜单

为了加速工作流程，插件还将核心功能添加到了骨架在不同模式下的右键菜单中。

*   **如何访问**: 选中骨架，进入 `对象模式`、`编辑模式` 或 `姿态模式`，然后单击右键，在菜单底部可以找到 `快速软骨绑定` 子菜单。

菜单中的可用操作会根据当前模式自动过滤，只显示最相关的命令。

## 3. 工具面板

在插件的偏好设置中（详情请见 [高级配置 -> 插件偏好设置](../advanced/preferences.md)），您还可以选择将面板同时显示在Blender左侧的"工具"栏中。
