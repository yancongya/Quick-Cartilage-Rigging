# 功能详解：生成FK绑定

在通过"骨骼细分"创建出骨骼链之后，下一步就是为这条链创建一套可用于动画的控制器。这就是"生成FK绑定"的功能。

FK，即正向动力学（Forward Kinematics），意味着您可以通过依次旋转父骨骼来带动子骨骼运动，是链状结构（如手臂、尾巴、鞭子）最基础和直观的控制方式。

---

## 功能解析

当您点击"生成FK绑定"时，插件会自动执行以下一系列操作：

1.  **创建控制器链**: 在您原有的"形变骨骼"（Deform Bones）旁边，平行创建一条"控制器骨骼"（Control Bones）链。这些新骨骼以 `ctr_` 作为前缀。

2.  **指定自定义图形**: 为所有控制器骨骼指定一个圆环（Circle）作为自定义图形，使它们在3D视图中清晰可见，易于点选。

3.  **设置父子关系与约束**: 插件会自动处理复杂的父子关系和约束设置，将形变骨骼与控制器骨骼关联起来。最终达成的效果是：当您在姿态模式下旋转一个控制器时，对应的形变骨骼也会随之旋转。

4.  **创建骨骼集合**: 为了方便管理视图，插件还会创建两个骨骼集合（Bone Collections）：
    *   `ctrl_{name}_all`: 包含所有新创建的控制器。
    *   `ctrl_{name}_first`: 只包含第一个控制器。
    这两个集合配合UI上的可见性按钮，可以快速切换控制器的显示状态。

---

## 使用方法

1.  确保您处于 **编辑模式 (Edit Mode)**。
2.  在刚刚由"骨骼细分"生成的骨骼链中，**选中任意一根骨骼**。插件会自动识别整条链。
3.  在插件面板的"绑定设置"区域，点击 **`2. 生成FK绑定`** 按钮。

## 预期结果

操作完成后，您会看到骨骼链周围出现了一圈圈的控制器。此时，FK绑定已经设置完毕。

同时，界面会弹出一个确认框，询问您 **"是否继续执行阻尼追踪?"**。这可以帮助您快速进入下一个，也是最后一个绑定步骤。
