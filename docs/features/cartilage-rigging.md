# 功能详解：软骨绑定（阻尼追踪）

这是"快速软骨绑定"流程的最后一步，也是最神奇的一步。它负责为骨骼链添加柔软、弹性的跟随效果，模拟真实的软骨或柔性物体表现。

这个效果的核心是Blender中的 **"阻尼追踪 (Damped Track)"** 约束。

---

## 功能解析

当您点击"生成软骨绑定"时，插件会智能地完成以下设置：

1.  **应用FK约束**: 首先，插件会为每一根"形变骨骼"添加一个"复制旋转 (Copy Rotation)"约束，使其精确地跟随对应"控制器骨骼"的旋转。这是实现FK控制的基础。

2.  **构建阻尼追踪链**: 这是实现"软骨"效果的关键。插件会从骨骼链的末端开始，依次为每一根形变骨骼添加一个"阻尼追踪"约束，使其"朝向"链条中的下一根骨骼。这构建了一个"你追我赶"的骨牌效应：
    *   `Bone.001` 追踪 `Bone.002`
    *   `Bone.002` 追踪 `Bone.003`
    *   ...以此类推。
    当您移动链条的根部时，这个力会像波浪一样逐级传递下去，从而产生自然的延迟和摆动效果。

3.  **设置统一驱动器**: 如果让用户手动去调节每一根骨骼上"阻尼追踪"的强度，那将是一场噩梦。因此，插件非常贴心地将所有"阻尼追踪"约束的强度（Influence）都通过驱动器（Driver）链接到了第一个控制器（`ctr_...001`）的一个自定义属性上——也就是您在UI上看到的 **"难崩系数"**。

    **这意味着，您只需要调节一个滑块，就可以控制整条骨骼链的"软硬"程度！**

---

## 使用方法

1.  确保您处于 **姿态模式 (Pose Mode)**。
2.  选中FK控制器链的 **第一根骨骼** (例如 `ctr_bone.001`)。
3.  在插件面板的"绑定设置"区域，点击 **`3. 生成软骨绑定`** 按钮。

## 预期结果

点击按钮后，所有约束和驱动器都会在后台自动配置完毕。您的软骨绑定现在已经完全可用！

您可以立即开始旋转FK控制器进行动画，并随时通过调节第一个控制器上的 **"难崩系数"** 来微调尾巴的弹性。
