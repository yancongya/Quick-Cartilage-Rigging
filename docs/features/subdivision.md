# 功能详解：骨骼细分

骨骼细分是使用本插件进行软骨绑定的第一步。它能将一根或多根选中的骨骼，快速转换为一整条骨骼链，为后续的FK绑定和软骨效果提供基础。

插件提供两种细分模式：`平均细分` 和 `斐波那契细分`。

---

## 1. 平均细分

`平均细分` 是最基础的细分方式，它会将选中的骨骼均匀地分割成指定数量的等长骨骼段。

*   **适用场景**: 当您需要一个长度均匀、弯曲弧度平滑的骨骼链时，例如卡通风格的飘带或需要均匀变形的物体。

### 参数

*   **段数 (Segments)**: 您希望将原骨骼分割成多少段。例如，输入 `10` 将会创建10根新的小骨骼。

### 使用方法

1.  在 **编辑模式** 下，选中您想细分的骨骼。
2.  在插件面板中设定好 **"段数"**。
3.  点击 **`平均细分`** 按钮。

---

## 2. 斐波那契细分

`斐波那契细分` 提供了一种更自然、更有机的细分方式。它会根据斐波那契数列生成一串长度不等的骨骼链，特点是 **根部密集，末端稀疏**（即骨骼长度由根到梢递增）。

*   **适用场景**: 模拟真实的尾巴、触手、辫子等结构。这些结构通常根部更灵活（需要更多骨骼来表现细节），而末端则摆动幅度更大。

### 参数

*   **段数 (Segments)**: 与平均细分相同，定义了新骨骼的总数。
*   **系数 (Coefficient)**: 控制斐波那契数列的增长率，从而影响骨骼长度的递增幅度。
    *   **系数越大**，骨骼链根部和末端的长度差异就越悬殊。
    *   **系数越小**，长度差异越不明显，效果越接近于平均细分。

### 使用方法

1.  在 **编辑模式** 下，选中目标骨骼。
2.  在插件面板中设定 **"段数"** 和 **"系数"**。
3.  点击 **`斐波那契细分`** 按钮。

---

## ✨ 高效技巧：自动执行

为了最大化效率，两个细分按钮都支持 **一键完成整个绑定流程** 的功能。

*   **如何触发**: 按住 `Alt` 键，然后点击 `平均细分` 或 `斐波那契细分` 按钮。

*   **效果**: 插件在执行完骨骼细分后，会自动继续执行 **"生成FK绑定"** 和 **"生成软骨绑定"** 的所有步骤。

这个功能非常适合快速搭建和测试效果，一步到位！
