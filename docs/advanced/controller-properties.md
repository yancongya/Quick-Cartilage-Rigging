# 高级配置：控制器属性

当您使用插件完成一套软骨绑定后，并不意味着所有参数都已固定。插件提供了一套强大的实时控制器属性，让您可以在姿态模式下随时调整绑定的表现，以满足动画需求。

---

## 如何访问控制器属性

1.  确保您处于 **姿态模式 (Pose Mode)**。
2.  在由本插件创建的骨骼链中，**选中第一个控制器骨骼** (例如 `ctr_bone.001`)。
3.  此时，在插件的主面板底部，会动态显示一个名为 **"控制器属性"** 的区域。

**重要提示**：所有这些属性都由第一个控制器统一管理。调整它的属性，会影响整条骨骼链。

---

## 可用属性详解

### 难崩系数 (Damped Track Influence)

这是决定动画"感觉"的最核心的属性。它控制着整条骨骼链的"软硬"程度或"弹性"。

*   **作用**: 实时调节阻尼追踪约束的强度。
*   **效果**:
    *   **值接近 `1.0`**: 骨骼链非常"硬"，几乎没有延迟，表现得像一根棍子，接近普通的FK链。
    *   **值接近 `0.0`**: 骨骼链非常"软"，延迟和摆动幅度最大，表现得像一根柔软的绳子。

您可以把它想象成在控制一根尾巴的硬度：数值越高，越像一根钢筋；数值越低，越像一根湿面条。

### 圆环缩放 (Circle Scale)

这是一个便利的显示工具，用于一次性调整整条链上所有FK控制器（圆环）的大小。

*   **作用**: 方便您根据模型比例和视图需要，放大或缩小控制器，以获得更整洁的视图或更方便的选择。

### 控制骨骼可见性

在处理复杂场景或进行精细动画时，管理视图中的元素非常重要。为此，插件提供了两个按钮来快速切换控制器的可见性。

*   **`控制骨骼` 按钮**: 按下此按钮，可以 **切换显示/隐藏** 整条链上的 **所有** 控制器骨骼。

*   **`控制根骨` 按钮**: 按下此按钮，可以 **切换"独显"模式**，即只显示链条的 **第一个** 控制器，并隐藏其余所有控制器。这个功能在制作大幅度的整体摆动动画时尤其有用，可以避免被过多的控制器干扰。
